## 2. ЛОГОТИП

### Структура логотипа
Логотип состоит из двух обязательных элементов:
1. **Графический знак (эмблема)** — геральдический символ в форме архитектурного строения с растительными элементами. Цвет: «Золотой песок» (`#B39168`).
2. **Текстовый блок** — название организации. Основное название выделяется цветом «Изумруд заботы» (`#00685B`), подстрочный текст — тёмно-серый или белый (в зависимости от фона).

### Варианты ориентации
- **Горизонтальная** — эмблема слева, текст справа. Используется как основная.
- **Вертикальная** — эмблема сверху, текст снизу. Допускается при ограниченной ширине.

### Цветовые версии логотипа
| Вариант | Применение |
|---------|-----------|
| Полноцветный на белом фоне | Основной вариант |
| Инверсия (светлый лого на тёмном фоне) | Тёмные и цветные поверхности |
| Монохромный | Печать с ограниченной палитрой |

### Охранное поле (защитная зона)
Минимальное расстояние вокруг логотипа от других объектов = высота буквы «Х» логотипного шрифта. Ничего не должно заходить в эту зону.

### Недопустимое использование логотипа
- Изменять пропорции или растягивать
- Менять фирменные цвета на произвольные
- Добавлять тени, обводки, градиенты
- Поворачивать логотип
- Размещать на пёстром/нечитаемом фоне
- Использовать фрагменты логотипа отдельно

## 3. ЦВЕТОВАЯ ПАЛИТРА

### Основные фирменные цвета

#### 🟢 «Изумруд заботы» — Основной цвет бренда
```
HEX:     #00685B
RGB:     R0  G104  B91
CMYK:    C96  M36  Y68  K24
PANTONE: 568 C
```
**Применение:** фоны, заголовки, акцентные блоки, кнопки, иконки, основные элементы интерфейса.

#### 🟡 «Золотой песок» — Акцентный цвет
```
HEX:     #B39168
RGB:     R179  G145  B104
CMYK:    C20  M35  Y60  K15
PANTONE: 6008 C
```
**Применение:** декоративные акценты, разделители, элементы логотипа, цифры разделов, вертикальные полосы.

### Дополнительные рабочие цвета (не указаны в палитре, но наблюдаются в материалах)
```
Белый:           #FFFFFF  — текст на тёмных фонах, фон документов
Тёмно-серый:     #333333 (приблизительно) — основной текст документов
Светло-серый:    #F5F5F5 (приблизительно) — нейтральные блоки, фоны
```

### Правила работы с цветом
- Основной зелёный (`#00685B`) используется как доминирующий фирменный цвет.
- Золотой (`#B39168`) — только как акцент, не как основной фон больших площадей.
- Белый фон является базовым для документов и презентаций.
- **Запрещено** использовать произвольные цвета вне палитры для фирменных элементов.

**Область применения:** деловые документы, переписка, презентации, рекламные материалы, печатные и цифровые носители, веб-интерфейсы.

**Поддерживаемые языки:** казахский | русский | английский

| Начертание | CSS / Font-weight | Применение |

## 5. ГРАФИЧЕСКИЕ ЭЛЕМЕНТЫ

### Система сплит-блоков (Split-blocks)

Основной визуальный приём айдентики — **модульные сплит-блоки** с асимметричными скруглениями.

**Форма:** прямоугольник с меньшими скруглениями слева и бо́льшими скруглениями справа.
```css
/* CSS-реализация */
border-radius: 8px 40px 40px 8px;   /* top-left, top-right, bottom-right, bottom-left */
/* Точные значения могут варьироваться в зависимости от размера блока */
```

**Философия формы:**
- Меньшее скругление слева → устойчивость, структурность
- Большее скругление справа → открытость, дружелюбие, движение вперёд

**Цвета сплит-блоков:** «Изумруд заботы» (`#00685B`) и «Золотой песок» (`#B39168`) как основные. Белый — для светлых блоков.

### Иконический элемент — стрелка
В кружке с обводкой используется стрелка вправо «→» как маркер пунктов и навигационный элемент. Цвет обводки — «Золотой песок».

### Фирменный паттерн
Повторяющийся орнаментальный паттерн на основе геральдических элементов логотипа. Используется как фоновая текстура на фирменной продукции (ткани, упаковки, сувениры).

## 6. ПРАВИЛА ВЁРСТКИ И МАКЕТИРОВАНИЯ

### Принципы построения макетов
- Вёрстка строится на сетке с чёткими полями
- Используются сплит-блоки как универсальный модуль
- Логотип всегда в охранном поле
- Заголовки набираются шрифтом **CAPS** только в особых акцентных элементах

### Корпоративные материалы (применение стиля)
| Носитель | Особенности |
|----------|------------|
| Фирменный бланк | Логотип вверху, название шрифтом Fira Sans Condensed, текст — Montserrat |
| Визитная карточка | Горизонтальная/вертикальная, фирменные цвета |
| Конверт | Логотип, фирменный зелёный |
| Папка | Фирменный паттерн или однотонный зелёный |
| Презентация | Сплит-блоки, фирменные цвета, Montserrat |
| Бейдж | Фото, имя, должность в фирменном стиле |

### SMM (социальные сети)
- Аватарки: логотип на зелёном фоне, квадратный формат
- Instagram: сплит-блоки, фирменные цвета
- YouTube: обложки с логотипом и фирменной типографикой
